Portfolio

Not sure what to eat for dinner? That's Recipeasy! Pick one of the randomly-generated recipes or search for something new! Compare the ingredients you have on hand against the recipe, then find a grocery store near you. Keep a personal diary about your entries - make it again? (API key currently deactivated. Video of funcionality soon to come!)

Glamazon is a CLI mockup of a database interaction using Javascript and MySQL. Instructions and a video of it functioning are in this Github readme.

MTG Host is a Magic: The Gathering companion app that tracks life totals and tokens players control using a MySQL server and Socket to keep each unique client updated.

Giphy search is a simple GIF search engine made using jQuery and Ajax. Select one of the hard-coded examples for an array of ten gifs and ratings, or use the search bar to add your own buttons.
